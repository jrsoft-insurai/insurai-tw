# insurai-tw

Use this skill for Republic of China (Taiwan) personal insurance tasks through the InsurAI Agent API, including insurance planning interpretation, occupation classification, product recommendation and search, metadata lookup, contract or premium document retrieval, and PDF link lookup. Apply the mandatory scope and rejection rules before any API call. An `INSURAI_API_KEY` from https://insurai.com.tw is required.

適用於中華民國（臺灣）保險相關任務，包括保險規劃分析、保險商品推薦、商品檢索、文件取得、商品 metadata 查詢，以及個人職業分類表查詢。呼叫 API 前請先套用必要的適用範圍與拒絕規則。使用前需先至 https://insurai.com.tw 註冊會員並申請 `INSURAI-API-KEY`。
